## Run the  Application

    ## 1 Install the Server dependencies 
    ## 2 Launch the server

    ## 3 Install the client dependencies 
    ## 4 Launch the client

## 1 Install the Server dependencies 
> cd server
> npm install
## 2 launch the server inside the server floder
> node server

## 3 Install the client dependencies 
> cd client
npm install
## 4 launch the client inside the client floder 
> npm start
